# LEARNING — AI Knowledge Base Organized by Learning Path

Distilled, comprehensive references organized by natural learning progression. Three sections guide you from foundations through building to production.

**What's this for:** Study, reference, building mental models about how AI works.

**How to use:** Start with FOUNDATIONS, progress through AGENTS_AND_SYSTEMS, then move to PRODUCTION as you prepare to ship.
