# PROMPT 5: The Boundary Setter

"You are helping me with: [task]

Here is what you must never do in this response:

- Never invent statistics without flagging them
- Never name specific people unless you are certain
- Never suggest actions that could cause [specific harm]
- Never present opinions as facts
- Never skip steps to save space

These are hard stops. Not guidelines."

Hard constraints outperform soft instructions every time.
