# PROMPT 3: The Source Challenger

"Complete this task: [task]

After your response, I will ask you one question:

'Where does each key claim in this response come from?'

Answer that question now, before I ask it.

For every claim: cite the type of source, the approximate time period, and your confidence level.

If you can't source it, flag it."

Sourcing prompts cut fabricated citations by 67% in NVIDIA's workflows.
