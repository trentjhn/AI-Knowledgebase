## Workflow

**When building something:**
1. Find the relevant playbook in **Playbooks/**
2. Grab prompt templates from **Prompt-Catalog/**
3. Reference back to **LEARNING/** for technical depth if needed
4. Use **Specs/** as examples of how to structure complex projects

---

**Philosophy:** These are tested, practical guides. Copy liberally. Adapt as needed. Keep them up-to-date as you learn new approaches.
